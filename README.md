# Bovine-Software-Graphics

This repository contains the code for a standalone software graphics driver for game development. You can use it wherever, or however, you want. 
It is written to target C89 for maximum interoperability across operating systems with the minimum operating system requirement officially being Microsoft Windows 2000. Older operating systems *may* work but are untested.

The header-file for this driver is ambiguous for the purpose of reusability within the larger Bovine Game Engine project.
